World Wide Web

Vznik WWW a principy

World Wide Web, ve volném překladu „celosvětová pavučina“, je označení pro aplikace internetového protokolu HTTP. Je tím myšlena soustava propojených hypertextových dokumentů. Služba se zrodila se v roce 1989 v evropském vědeckém centru CERN v Ženevě díky Timu Berners-Lee.

Za hypertext označujeme takový text, který obsahuje propojení na jiné texty pomocí tzv. hypertextových odkazů. Ve službě WWW jsou realizovány jako tzv. URL odkazy. Služba je založena na spolupráci WWW klientů s WWW serverem. Prvním komerčním prohlížečem webových stránek byl na počátku 90. let minulého století Mosaic. Při jejich spolupráci fungují následující mechanismy:

  • komunikační protokol HTTP, pomocí něhož WWW klient a WWW server komunikují
  • jazyk HTML, definující formát WWW stránek, které server vrací klientovi.

W3C je mezinárodní konsorcium jehož členové společně s veřejností vyvíjejí webové standardy pro WWW. Konsorciu předsedá Tim Berners-Lee.

Webový server

1. Jakou roli plní webový server? Uveďte příklady webových serverů.

Server je bezobslužný program, který přijímá a obsluhuje požadavky klientů. Na vlastnostech tohoto programu závisí, co server dovede a jaký má výkon. Webový server je připojen k počítačové síti a přijímá požadavky v souladu s protokolem HTTP (HTTP Request). Tyto požadavky vyřizuje a počítači, který požadavek vznesl, vrací odpověď.

Příklady webových serverů:

  • Apache
  • NGINX
  • Microsoft Internet Information Server
2. Jak se nazývá program označovaný jako webový klient? Co je jeho úkolem? Vyjmenujte nejpoužívanější webové klienty.

Jinak také webový prohlížeč, komunikuje s uživatelem a webovým serverem. Podle pokynů uživatele ze serverů získává data, zformátuje je a zobrazí na obrazovce uživatele.

Nejpoužívanější webové klienty:

  • Google Chrome
  • Safari
  • Mozilla Firefox
  • Microsoft Edge
  • Opera
3. Jak probíhá komunikace mezi webovým klientem a webovým serverem?
Komunikují spolu pomocí protokolu HTTP. Jazyk HTML definuje formát stránek, které server vrací klientovi.
4. Co je HTTP?
HyperText Transfer Protocol, internetový protokol používaný k přenosu souborů mezi klientem a serverem. Funguje na principu požadavek - odpověď.
